# 产品概览

同窗是一个会看见、会行动、会与你协作的桌面 AI 伙伴，让电脑成为有生命感的工作空间。

它不是只会聊天的助手，而是围绕本地视觉控制循环运行：

```text
观察屏幕 -> 理解任务 -> 执行一步操作 -> 再次观察
```

因此同窗可以在浏览器、编辑器、文档、网页、桌面应用和代码工作区里通过真实鼠标、键盘、语音和后台 Code Agent 工作流推进任务。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| 看见桌面 | 截图观察、视觉推理，并支持截图后端回退。 |
| 多屏幕操作 | 理解屏幕元数据，可以跨多个显示器操作。 |
| 操作电脑 | 点击、拖拽、滚动、快捷键、输入文本、打开浏览器、读取网页或文档。 |
| 与人协作 | 悬浮助手提供任务输入、停止控制、实时日志、设置窗口和伴随推荐。 |
| 听见并回应 | 支持实时 ASR、TTS、本地唤醒词、VAD、回声消除和录音状态提示。 |
| 中英文支持 | 提供中英文界面文案和双语文档路径。 |
| 后台 Code Agent | 通过 Codex、Claude、Kimi、Qwen、CodeBuddy 等 provider 在后台执行代码与自动化任务。 |
| 跨平台 | 提供 macOS 和 Windows 平台适配。 |
| 模型灵活 | 通过 `base_url`、`api_key`、`model_name` 接入 OpenAI-compatible 模型服务。 |
| 易于集成 | 提供 GUI、CLI、Python API、测试和示例。 |

## 当前状态

同窗 2.0 仍处于 beta 阶段。核心架构已经拆分为 GUI、平台适配、语音、Code Agent、模型调用与 runner 模块。模型适配、界面体验、打包发布和跨平台验收仍在快速迭代中。
